Vendor Project Plugin
=====================

Описание
--------

Версия плагина для управления composer-зависимостями напрямую из него, а не из Ignet Dev Toolkit.
В базовой комплектации содержит паке DT Core с главными классами.
Может содержать другие пакеты из набора DT, и прочие composer пакеты.

Развёртывание
-------------

> composer install
или
> composer update

Если происходит ошибка авторизации, установить свои данные в composer выполнив:
> composer config http-basic.bitbucket.org username password

Кастомизация
------------

Для добавления новых пакетов DT указать адрес репо в директиве repositories, и зависимость в директиве require.
Для нативных composer пакетов просто указать зависимость.

После добавления новых пакетов не забыть обновить автозагрузку выполнив:
> composer update

----------------------------------------

желательно с указанием версий и ссылок на документацию.

Все хитрое, сложное, важное, интересное стоит здесь описать.
Этот файл пишется для разработчиков или клиентов способных делать что-либо с кодом.
